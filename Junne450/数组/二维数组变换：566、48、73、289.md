## 二维数组变换：566、48、73、289

###566.[重塑矩阵](https://leetcode-cn.com/problems/reshape-the-matrix/)

```java
class Solution {
    public int[][] matrixReshape(int[][] mat, int r, int c) {
        int m = mat.length;
        int n = mat[0].length;
        int p=0,q=0;
        if(r*c!=m*n){
            return mat;
        }
        int[][] res = new int[r][c];
        for(int i=0;i<m;i++){
            for(int j=0;j<n;j++){
                res[p][q]=mat[i][j];
                if(q<c-1){
                    q++;
                }else{
                    p++;
                    q=0;
                }
            }
        }
        return res;
    }
}
```

原数组按顺序遍历，用p、q来控制对应变换后数组的下标位置。
