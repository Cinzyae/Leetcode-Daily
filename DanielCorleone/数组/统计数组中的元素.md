## 统计数组中的元素

### [错误的集合](https://leetcode-cn.com/problems/set-mismatch/)

#### 思路

- HashSet去重配合求和公式。

```java
class Solution {
    public int[] findErrorNums(int[] nums) {
        Set<Integer> set = new HashSet<>();
        int sum = 0;
        int setNum = 0;
        for (int i = 0; i < nums.length; i++) {
            sum += nums[i];
            if (!set.contains(nums[i])) {
                setNum += nums[i];
                set.add(nums[i]);
            }
        }
        int n = nums.length;
        int s = (n + n * n) / 2;
        return new int[]{sum - setNum, s - setNum};
    }
}
```
