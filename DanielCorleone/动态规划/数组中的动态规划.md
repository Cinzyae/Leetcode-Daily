## [509. 斐波那契数](https://leetcode-cn.com/problems/fibonacci-number/)

模板题

```java
class Solution {
    public int fib(int n) {
        if (n <= 1) return n;
        int f0 = 0, f1 = 1, f2 = 1;
        for (int i = 2; i < n; i++) {
            f0 = f1;
            f1 = f2;
            f2 = f0 + f1;
        }
        return f2;
    }
}
```

## [70. 爬楼梯](https://leetcode-cn.com/problems/climbing-stairs/)

```java
class Solution {
    public int climbStairs(int n) {
        if (n <= 3) return n;
        int f0 = 0, f1 = 1, f2 = 2;
        for (int i = 2; i < n; i++) {
            f0 = f1;
            f1 = f2;
            f2 = f0 + f1;
        }
        return f2;
    }
}
```
